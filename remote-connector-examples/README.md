# Remote Connector Examples

This repository contains example connectors for RemoteTask, demonstrating various authentication patterns and tool implementations.

## Examples

- **connector-example** - Complete example with async tools, webhooks, and data processing
- **connector-example-api-key** - API Key authentication example
- **connector-example-basic** - Basic Auth authentication example
- **connector-example-custom** - Custom authentication example
- **connector-example-oauth** - OAuth2 authentication example
- **connector-example-session** - Session-based authentication example

## Getting Started

Each example connector is a standalone package. Navigate to the specific example directory and follow its README for setup instructions.
